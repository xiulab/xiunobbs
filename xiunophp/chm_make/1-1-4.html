关于 URL 格式

XiunoPHP 除了支持原生的 URL 格式，还支持以下两种伪静态 URL 格式，前提是做好 URL-Rewrite，具体方法，请参看  4.1.4: URL-Rewrite。

http://domain.com/user-login.htm
http://domain.com/user-login.htm?uid=123

http://domain.com/user/login
http://domain.com/user/login?uid=123

获取参数的方法：

<?php

include './xiunophp/xiunophp.php';
print_r($_REQUEST);
/*
Array(
	0 => 'user',
	1 => 'login',
	'uid' => 123,
)
*/

// 获取参数
$a = param(0);		// "user"
$c = param(1);		// "login"
$uid = param('uid', 0); // 123	第二个参数为默认的数据类型和值，用来安全获取参数

?>
